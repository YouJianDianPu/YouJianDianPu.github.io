列表

无序列表使用星号、加号和减号作为列表标记

  • Red
  • Green
  • Blue
    等同于
  • Red
  • Green
  • Blue
    等同于
  • Red
  • Green
  • Blue
    有序列表则使用数字接着一个英文句点
  1. Red
  2. Green
  3. Blue

分割线

你可以在一行中用三个以上的星号、减号、底线来建立一个分割线，行内不能有其他的东西。

强调

使用星号和底线作为标记强调字词的符号，被一个星号或者底线符号包围的字词会被转成<em>标签，被两个星号或者底线包围的字词则会被转成<strong>

代码

标记一小行代码，可以用反引号包起来，例如
