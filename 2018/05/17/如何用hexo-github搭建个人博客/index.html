如何用hexo+github搭建个人博客

Posted on 2018-05-17 | In hexo+github

第一步:

下载安装git和node.js

第二布:

安装hexo
使用npm install hexo -g命令

第三步:

初始化hexo
使用hexo init命令

第四步:

配置_config.yml
找到Site处的title更改博客名字, author更改作者名字; 找到deploy处的type填入git, repo处填入你的github个人博客的仓库地址

第五步:

使用hexo new "page"来创建文档, 在文档上即可写博客, 支持Markdown语法

第六步：

使用npm install hexo-deployer-git --save安装自动部署工具

第七步：

博客书写完毕后, 使用hexo clean && hexo g && hexo d即可发布到github上, 浏览器输入相应仓库地址即可访问, 本地环境可用hexo s命令，浏览器默认地址是localhost:4000
